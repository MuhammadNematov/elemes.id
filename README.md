## 📖 О проекте

Этот проект представляет собой современный, адаптивный лендинг для сервиса доставки готовой еды. Сайт разработан для привлечения клиентов, демонстрации меню, преимуществ сервиса и удобного оформления заказа.

**Посмотреть демо:** [Ссылка на ваш GitHub Pages](https://ваш-аккаунт.github.io/ваш-репозиторий)

## ✨ Особенности

- **Адаптивный дизайн** — корректно отображается на телефонах, планшетах и десктопах
- **Семантическая верстка** HTML5
- **Стилизация Tailwind CSS** — быстрая и гибкая разработка
- **Оптимизация изображений** — быстрая загрузка блюд
- **Кросс-браузерная совместимость**
